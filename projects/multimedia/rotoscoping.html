FINAL OUTPUT

The Rotoscoping animation was created using Adobe After Effects 2023. I used the Roto Brush tool to isolate the dancer frame by frame, separating the subject from the original video.

After isolation, I applied the Fill effect to stylize the character with solid colors. This added a graphic and animated feel to the visuals, making the movement stand out more clearly.

To finish the edit, I exported the sequence into CapCut. There, I added animated lyrics and synced visual effects to the rhythm of the music. This combination helped create a polished, expressive, and dynamic final output.

Tools Used

In Adobe After Effects 2023

RotoBrush Tool

The Roto Brush tool allows you to separate a subject from its background by painting strokes over the desired object. It‚Äôs useful for rotoscoping because it enables a semi-automated way to create complex cutouts frame by frame.

Pen Tool

Used to create precise masks and refine edges where the Roto Brush had limitations. It also allows custom shapes and outlines for artistic control in animations.

Transform Properties + Stopwatch

Using keyframes with transform properties allows motion such as scaling, rotating, and translating. This brings life to static elements.

Freeze Feature

Locks in Roto Brush selections across all frames. Prevents AE from recalculating changes during editing ‚Äî ensuring consistency.

Tools Used
